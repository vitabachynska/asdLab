##№Інструкція
##Технічні вимоги
**Java**: JDK 17+
**Бібліотеки**: Jackson для роботи з JSON.
**База даних**: файл university_data.json (створюється автоматично).

##Як запустити
**IDE**: Відкрийте проєкт, додайте Jackson до бібліотек та запустіть ui.Main.
**Початок**: У меню «Робота з університетом» створіть заклад (назва, адреса).
**Доступ**: Для редагування -> Менеджер (пароль: 1234), для зміни паролів "Адміністратор" (пароль: 4321). Все інше -> Гість (пароль не потребує). Паролі захищені за допомогою алгоритму SHA-256.
**Структура**: Спершу створіть Факультет, потім у ньому Кафедру, і лише тоді додавайте Студентів/Викладачів. Тестові дані завантажаться з файлу.
**Пошук**: Використовуйте підменю студентів/викладачів для фільтрації за ПІБ чи курсом.
**Збереження**: Дані зберігаються автоматично після кожної дії. Виходьте через "0", щоб уникнути помилок.

##Проблеми
**Не зберігає/не виводить дані**: Перевірте шлях до файлу в FileHandler.java (змініть на свій абсолютний).
