# Інструкція

## Технічні вимоги
* **Java:** JDK 17+
* **Бібліотеки:** Jackson для роботи з JSON
* **База даних:** файл "university_data.json"

## Як запустити
1. **IDE:** Відкрийте проєкт, додайте Jackson до бібліотек та запустіть "ui.Main"
2. **Початок:** У меню «Робота з університетом» створіть заклад (назва, адреса)

## Доступ та безпека
* **Менеджер:** пароль "1234"
* **Адміністратор:** пароль "4321"
* **Захист:** Усі паролі захищені алгоритмом SHA-256

## Порядок дій
* **Структура:** Спершу створіть Факультет, потім Кафедру, і лише тоді додавайте Студентів/Викладачів
* **Збереження:** Дані зберігаються автоматично після кожної дії. Виходьте через "0", щоб уникнути помилок

## Вирішення проблем
**Не зберігає/не виводить дані:** Перевірте шлях до файлу в "FileHandler.java" та замініть його на свій актуальний
